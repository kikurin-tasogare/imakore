# imakore PWA

「今、頭にあることを、そのまま残す」ためのiPhone向けWebアプリです。

## 特徴

- 「残す」「振り返る」の2画面だけ
- 繰り返し現れる言葉を端末内で自動分析
- テーマから関連する過去の思考を絞り込み
- 全文検索と、すぐ試せる削除可能なサンプル
- 思考、作成日時、更新日時を端末内に保存
- JSON形式でのバックアップと復元
- 入力途中の文章も端末内に保持
- ホーム画面への追加とオフライン利用に対応
- ダークモード、Dynamic Type相当の文字拡大、VoiceOverに配慮

データはブラウザのローカルストレージに保存されます。アカウント登録や外部送信は行いません。SafariのWebサイトデータを消去すると、保存した思考も削除されます。

## 開発

Node.js 22.13以降とpnpmを使用します。

```bash
pnpm install
pnpm dev
pnpm build
pnpm test
```
